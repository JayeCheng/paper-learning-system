# Knowledge Graph

This directory keeps human-readable topic maps. The future machine-readable graph
contract is defined by `schemas/knowledge_graph.schema.json` and should be published
through `data/public/`.

## Topic Maps

- [GPU / Graphics](gpu_graphics.md)
- [Computer Architecture](architecture.md)
- [LLM Agent / RAG](llm_agent.md)
- [Cognitive Science](cognition.md)
- [Sociology](sociology.md)

## Conventions

- Prefer stable node names.
- Link papers to concepts, methods, benchmarks, and open questions.
- Record uncertainty instead of forcing weak edges.
- Keep generated JSON separate from these Markdown maps.
