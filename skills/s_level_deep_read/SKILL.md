# S-Level Deep Read

## Purpose

Create a deep-read note for a high-value paper.

## Inputs

- selected paper metadata
- existing topic notes
- curriculum context

## Outputs

- Markdown note under `deep_read/<track>/`
- knowledge graph update proposal
- reading status update proposal

## Boundaries

- Do not store private annotations.
- Do not rewrite unrelated notes.
- Do not create new state formats without schema updates.
