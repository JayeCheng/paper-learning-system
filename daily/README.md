# Daily Reports

Daily reports are the human-readable paper radar archive.

The intended pair is:

- `daily/YYYY/MM/YYYY-MM-DD.md` for long-term reading
- `daily/YYYY/MM/YYYY-MM-DD.json` for structured consumers

Markdown should stay useful to humans. Frontends should start from
`data/public/latest.json` or `data/public/daily_index.json`.
