# Curriculum Roadmap

The curriculum engine balances new papers with classics.

## Tracks

- GPU / Graphics
- Computer Architecture
- LLM Agent / RAG
- Cognitive Science and Behavioral Science
- Sociology

## Weekly Rhythm

- Skim daily radar candidates.
- Select up to three deep reads per week.
- Connect deep reads to the knowledge graph.
- Promote enduring papers into classic routes.

## Review Questions

- What concept did this paper clarify?
- What should be read before and after it?
- Which knowledge graph edge changed?
- Is it worth adding to a classic-paper sequence?
